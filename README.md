### corepost-server
предоставляет HTTP API для admin, mobile, preboot и agent.
Сервис запускается через docker compose и хранит состояние устройств в локальной базе.
OpenAPI публикуется самим сервером и используется для интеграции клиентов.
Тесты покрывают регистрацию, блокировку, разблокировку, предзагрузочный доступ и agent polling.
